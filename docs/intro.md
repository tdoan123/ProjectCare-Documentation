# TeamYellow Documentation

**A Health Client Management platform for counsellors built with ASP.NET Core MVC**

---

## Overview

TeamYellow allows counsellors to subscribe to different service tiers (Free, Monthly, Yearly) and manage their practice. This project demonstrates clean architecture principles, PayPal integration, and ASP.NET Core best practices.

**Tech Stack:** ASP.NET Core 8.0 MVC • SQLite • Entity Framework Core • PayPal REST API • ASP.NET Core Identity

## Project Features

✅ Three subscription tiers (Free, Monthly $49.99, Yearly $499.99)  
✅ PayPal payment integration with sandbox testing  
✅ Role-based access control (Registered_Visitor → Free_Counselor → Paid_Counselor)  
✅ Auto-generated unique practitioner licence IDs  
✅ Subscription lifecycle management (Active, Cancelled, Expired)  
✅ Duplicate payment prevention
